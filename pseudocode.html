Med "pseudokod" menar vi en beskrivning av koden i prosa, men med strukturen av programkod. Det är ett beprövat sätt att organisera programmet utan att behöva följa reglerna för ett visst programspråk.

Vi använder nyckelorden while och if eftersom dessa är traditionsbundna och motsvarande ord på svenska ("medan", "om") saknar rätt magisk kraft.

secretNumber ← ett slumpat tal mellan 1 (inklusive) och 100 (inklusive)
while spelet inte är vunnet:
  number ← nästa gissning från användaren
  if number = secretNumber:
    vinn spelet
  if number < secretNumber:
    säg "gissa högre"
  if number > secretNumber:
    säg "gissa lägre"
